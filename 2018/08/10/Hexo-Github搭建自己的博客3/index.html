    ▏这个在段落前等于是空格

    各种三角 ▷ ◁ ▶ ◀ △▽▲▼

    ➥ 和 → 还有 ˉˉ↴ 以及 ➨ 起引导视线的作用

    ☛ 这个是点击

    ☑ 和 ✔ 是已经选择，OK

    ✘ 是错误，删除，NO

    ✄ 是剪切

    ❐ 这个是复制粘贴

    ✚ 是添加，补充，选项

    ☷ 是列表，调整格式

    ① ② ③ ④ ⑤ ⑥ ⑦ ⑧ ⑨ ⑩ 这样的数字是不是很酷
